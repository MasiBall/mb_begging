# mb_begging
Simple 'beg money' resource for FiveM

# Description
This resource allows players to beg for some cash from NPCs

# Dependencies
- ESX

# Installation Guide
1. Download the resource from Github
2. Drag n' drop the folder to your `resources` folder
3. Rename the folder to `mb_begging`
4. Add `ensure mb_begging` to your server.cfg file
5. Configure config.lua and server.lua files
6. Ensure the resource or restart your server

# Usage
- Press H to beg for money from the nearest NPC
- After the cooldown, you are able to beg again.
